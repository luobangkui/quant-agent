# POC 使用说明（基于 ali.csv）

本仓库提供 `data/ali.csv` 示例行情数据，可用于快速验证量化核心雏形。

## 数据位置
- `data/ali.csv`：阿里巴巴（HK.09988）日线历史数据，含收盘价、换手率等字段。

## 快速开始
1. 安装依赖：`pip install -r requirements.txt`
2. 运行 POC 脚本：`python scripts/poc_ali.py --csv data/ali.csv --short 5 --long 20`
3. 终端会输出数据范围、样本量以及基于双均线的总收益、年化、夏普与回撤等指标。

## 目录说明
- `core/data/loaders.py`：CSV 行情加载器，负责解析时间戳与数值字段。
- `core/strategies/momentum.py`：简单双均线动量策略与绩效计算。
- `scripts/poc_ali.py`：POC 入口，串联数据加载、策略运行与指标打印。

如需替换数据或调参，可修改命令行参数 `--csv`、`--short`、`--long`。
